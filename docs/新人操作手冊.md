# DistanceViewer 新人操作手冊

本手冊給第一次接觸專案的使用者，目標是讓你在 **10 分鐘內**完成安裝並開始查詢島嶼座標。

---

## 1. 這個工具可以做什麼？

DistanceViewer 提供兩種使用方式：

1. **Web 版（推薦）**：在瀏覽器操作，適合大多數新手。
2. **桌面版（Desktop）**：使用 Matplotlib 互動視窗，適合需要進階手動編輯點位者。

核心功能包含：

- 依「資源 / 神蹟」篩選島嶼點位。
- 計算點位間距離（透過三角網邊線）。
- 手動新增、移動、刪除點位。

---

## 2. 先備條件

請先安裝：

- **Python 3.10 以上**（建議 3.11）
- 能使用終端機（Windows PowerShell、macOS Terminal、Linux shell）

> 資料來源檔案 `Ikariam島嶼_New_20260323.xlsx` 已在專案中，請勿任意更名或刪除。

---

## 3. 第一次安裝（只做一次）

在專案根目錄執行：

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Windows 使用者啟用虛擬環境

```powershell
.venv\Scripts\activate
```

---

## 4. 啟動與使用 Web 版（推薦）

### 4.1 啟動伺服器

```bash
flask --app api.index run --debug
```

看到類似訊息後代表成功：

- `Running on http://127.0.0.1:5000`

### 4.2 開啟網頁

在瀏覽器打開：

- <http://127.0.0.1:5000>

### 4.3 基本操作流程

1. 在篩選區選擇 **資源**、**神蹟**（可留空表示不限制）。
2. 設定座標範圍 `x_min/x_max/y_min/y_max`（有效範圍 0~100）。
3. 送出查詢後，圖上會顯示符合條件的島嶼。
4. 可以在圖上互動編輯點位，並切換是否顯示邊線與標籤。

---

## 5. 啟動與使用桌面版（進階）

執行：

```bash
python DisViewer.py
```

桌面版常用操作：

- **左鍵**：新增點 / 變更點顏色
- **右鍵**：刪除點
- **拖曳**：移動點
- **Ctrl+Z / Ctrl+Y**：復原 / 取消復原
- 可輸入「資源 / 神蹟」條件自動加入符合點位

---

## 6. 常見問題排除

### Q1：啟動時出現 `Excel file not found`

- 請確認你在專案根目錄啟動程式。
- 確認檔名仍為 `Ikariam島嶼_New_20260323.xlsx`。

### Q2：`ModuleNotFoundError` 套件缺失

請重新安裝依賴：

```bash
pip install -r requirements.txt
```

### Q3：Web 查詢失敗，顯示座標範圍錯誤

- `x_min <= x_max`、`y_min <= y_max`
- 所有值必須是整數，且介於 0~100

### Q4：看不到中文字型（桌面版）

- 若系統沒有 `DFKai-SB`，程式會自動切換英文字型與英文提示，屬正常行為。

---

## 7. 給新人的操作建議

- 先用 **Web 版**熟悉篩選與點位概念。
- 需要大量手動調整點位時，再改用 **桌面版**。
- 每次更新依賴前，建議先備份自己的環境或使用新的虛擬環境。

祝使用順利！
